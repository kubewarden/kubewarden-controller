# Kubewarden

[Kubewarden](https://kubewarden.io) makes it very easy to write and distribute Kubernetes admission and mutation policies using your preferred languages and frameworks, and to publish them through well-known distribution methods such as OCI-artifact-enabled image registries.

It is powered by [WebAssembly](https://webassembly.org/), so Kubewarden policies are architecture and operating system agnostic.

Discover many policies contributed and maintained by the community in [ArtifactHub](https://artifacthub.io/).

Download policies or build your own once, and run them everywhere, no matter what the system is.
